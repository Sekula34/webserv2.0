Basic checks

Using telnet, curl, prepared files, demonstrate that the following features work properly:

GET, POST and DELETE requests should work.

GET: curl localhost:9494/ With this command, you'll GET the index page.

POST: curl --data-binary "@image.png" http://localhost:9494/?image.png With this command, you'll POST (upload) a file to the server.

DELETE: curl -X "DELETE" 'http://localhost:9494/uploads/image.png' With this command, you'll DELETE the file on the server.

UNKNOWN requests should not result in a crash.

UNKNOWN request: curl -X WATEVERIWANT localhost:9494/ -v With this command, you'll send a WATEVERIWANT request (invalid method) to the server.

UNKNOWN request: curl localhost:9494/idontexist.html With this command, you'll send a GET request to the server for an unexisting file.

Upload some file to the server and get it back.

Upload: curl --data-binary "@image.png" http://localhost:9494/?image.png

Download: curl http://localhost:9494/uploads/image.png -o download.png
